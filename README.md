# ロボットシステム学2018課題２

* 自作の対向二輪型ロボットにROSを導入
* 外部のマシンからstd_msgs/Int16型のメッセージを送ることによりロボットを操作
*回りたいの角度を入力、ロボットは回る

## 自作したロボット

*　Raspberry Pi 3B
*　arduino
* エンコーダー付けるモーターを使用
*クローラーを使用
